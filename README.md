# Monthly User Message Activity
SQL analysis of monthly user messaging activity and engagement metrics.

## Business Task
Analyze monthly messages sent to users and calculate the share of messages for each user.

## Tools
SQL, Joins, Aggregations, Window Functions, Date Functions, Views

## Key Insights
- The dataset shows monthly message activity for each user.
- The analysis calculates the percentage of messages each user received.
- First and last message dates show user activity during the month.
