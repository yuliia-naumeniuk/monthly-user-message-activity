# Monthly User Message Activity
SQL analysis of monthly user messaging activity and engagement metrics.

## Business Task
Analyze how many messages users send each month and calculate the share of messages per user.

## Tools
SQL, Joins, Window Functions, Aggregations, Date Functions

## Key Insights
- The dataset shows monthly message activity for each user.
- The analysis calculates the percentage of messages sent by each user.
- First and last message dates show when users were active during the month.
